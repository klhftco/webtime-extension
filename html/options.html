      Tracking mode

      Focused mode tracks only the active tab of the focused window. Visible windows mode tracks the active tab in each non-minimized browser window (useful for split-screen) and only counts a normalized domain once even if it appears in multiple windows. Chrome extensions cannot detect OS desktops, so for the most accurate tracking minimize windows or switch them to an untracked page (like the New Tab page) when moving them to another desktop.
